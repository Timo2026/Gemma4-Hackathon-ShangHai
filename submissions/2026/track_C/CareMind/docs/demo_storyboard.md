# CareMind Demo Storyboard
**时长目标**：40–50 秒
**演示用户角色**：林女士，照顾失智母亲已两年（虚构人物）
**演示患者昵称**：妈妈（虚构）
**演示设备**：iPhone，竖屏录制

---

## 整体叙事弧线

```
人的处境（5s）
→ 打开 App，说出今天（8s）
→ CareMind 帮你整理（8s）
→ 今晚知道做什么（8s）
→ 知道怎么回应她（8s）
→ 复诊材料一键生成（8s）
→ 收尾（5s）
```

---

## 分镜详情

### Scene 1｜人的处境（0–5s）

**画面**：黑底白字，逐字出现
**文案**：
> 照顾失智的妈妈，已经两年了。
> 每天发生的事，说不清楚，也记不住。

**配音**：无（纯字幕）
**背景音**：轻柔钢琴，低音量

---

### Scene 2｜打开 App，进入智能记录（5–13s）

**画面**：手机解锁 → 打开 CareMind → 进入「智能记录」Tab
**演示操作**：点击底部「智能记录」，页面展示输入框
**录制重点**：
- 页面标题「智能记录」
- 输入框占位符「今天妈妈有什么让你担心的事吗？」
- 用户开始输入（预设文字逐字出现或直接粘贴）

**演示输入文字**：
> 妈妈昨晚起来四次，今天一直说有人偷她的钱，晚饭只吃了几口。我也快撑不住了。

**配音**（可选）：
> "我把今天发生的事，用一句话说给它听。"

---

### Scene 3｜帮我整理（13–21s）

**画面**：点击「帮我整理」按钮 → 进度卡出现（4 步逐步完成）
**进度步骤文案**（屏幕上可见）：
- 整理今天发生的事 ✓
- 找出值得留意的地方 ✓
- 生成沟通建议 ✓
- 记住有用的照护方式 ✓

**配音**（可选）：
> "它帮我把混乱的照护日记，整理成清晰的记录。"

**录制重点**：进度卡动画（勾号逐步出现）

---

### Scene 4｜结构化记录 + 今晚值得留意（21–29s）

**画面**：结构化摘要卡 + 关注事项卡（需留意：夜间安全）
**录制重点**：
- 摘要行：睡眠「夜间起床 4 次」/ 行为「说有人偷钱」/ 饮食「只吃了几口」
- 关注事项卡：左侧橙色竖条 + 「今晚值得留意」
- 展开行动建议：「打开夜灯，清理床边障碍物」+ 「已处理」「做不到」按钮

**配音**（可选）：
> "今晚需要注意什么，可以做什么，都写清楚了。"

---

### Scene 5｜沟通话术（29–37s）

**画面**：InstantScriptCard 展示沟通建议
**录制重点**：
- 「不建议说」灰色区：「没人偷，你别乱想。」
- 「可以试着说」绿色区：「你是不是很担心？我陪你一起找找。」
- 播放按钮（可点击演示 TTS 播放）

**配音**（可选）：
> "遇到她说有人偷钱，我不知道怎么回应。现在有了。"

---

### Scene 6｜复诊材料（37–45s）

**画面**：切换到「复诊准备」Tab → 展示照护摘要卡 + 医生问题清单
**录制重点**：
- 「家属照护记录」标题
- 近 7 天摘要段落（已有数据）
- 「你可能想问医生的问题」清单（2–3 条可见）
- 「生成可复制摘要」绿色按钮

**配音**（可选）：
> "下周复诊，不用再靠回忆了。"

---

### Scene 7｜收尾（45–50s）

**画面**：CareMind Logo（HeartHandshake 图标）+ 产品定位语
**文案**：
> CareMind
> 帮家属把记忆变成医生能看懂的材料。
> 不替代医生，只做你身边的照护记录伙伴。

**背景音**：淡出

---

## 关键节奏控制

| Scene | 时间点 | 关键镜头 |
|---|---|---|
| 处境 | 0–5s | 白字渐显 |
| 记录输入 | 5–13s | 文字逐字出现 |
| 整理进度 | 13–21s | 勾号逐步点亮 |
| 结构化 + 关注 | 21–29s | 摘要行 + 关注事项卡 |
| 话术 | 29–37s | 绿色区「可以试着说」 |
| 复诊 | 37–45s | 摘要 + 导出按钮 |
| 收尾 | 45–50s | Logo + 定位语 |

---

## 演示前准备

1. 进入 App → 加载演示数据（点击「加载 3 分钟演示数据」）
2. 清除前一次输入，保持智能记录页干净
3. 手机勿扰模式开启
4. 字体大小使用默认（不放大也不缩小）
5. 屏幕亮度调至最高
