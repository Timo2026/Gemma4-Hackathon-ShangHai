# Demo 录制指南

---

## 录制前准备

### 设备设置
- [ ] 手机勿扰模式开启（清除通知）
- [ ] 屏幕亮度调至最高
- [ ] 字体大小：系统默认（不放大）
- [ ] 深色模式：关闭
- [ ] 状态栏时间设置为整点（如 9:41），或录制后用编辑遮盖
- [ ] 关闭所有非必要 App，避免切换动画干扰

### App 状态
- [ ] 清除所有历史记录（设置页 → 重置演示数据）
- [ ] 点击「加载 3 分钟演示数据」，确认今日照护页有关注事项卡
- [ ] 智能记录页：确认输入框为空
- [ ] 复诊准备页：确认有摘要内容

## 录制工具推荐

| 平台 | 工具 | 说明 |
|---|---|---|
| iOS 原生 | 控制中心「屏幕录制」| 直接录，最简便 |
| Mac 镜像录制 | QuickTime + Lightning/USB | 更高清，适合横版剪辑 |
| 后期剪辑 | CapCut / DaVinci Resolve / 剪映 | 加字幕、配音、转场 |

## 录制脚本（操作步骤）

按分镜顺序执行，每步留 1–2 秒停顿，方便后期剪辑：

**Step 1**（0–5s）：字幕卡，不需要录制 App，后期制作

**Step 2**（5–13s）：
1. 手机在桌面，录制解锁动作
2. 点击 CareMind 图标，App 打开
3. 点击底部「智能记录」Tab（Home 图标旁）
4. 输入框页面出现，停顿 1s
5. 开始在输入框里粘贴或打字（推荐粘贴，更流畅）：
   > 妈妈昨晚起来四次，今天一直说有人偷她的钱，晚饭只吃了几口。我也快撑不住了。

**Step 3**（13–21s）：
1. 点击「帮我整理」绿色按钮
2. 等进度卡出现，4 个步骤依次勾选完成
3. 停顿直到所有步骤打勾

**Step 4**（21–29s）：
1. 向下滚动，结构化摘要卡出现（睡眠/行为/饮食/照护者）
2. 继续滚动，今晚值得留意卡出现
3. 点击关注事项卡标题，展开（展示行动项）
4. 停顿展示「已处理」「今天做不到」按钮

**Step 5**（29–37s）：
1. 继续向下滚动至沟通话术卡
2. 停顿展示「不建议说」和「可以试着说」两个区块
3. （可选）点击「播放这句话」展示 TTS 功能

**Step 6**（37–45s）：
1. 点击底部「复诊准备」Tab
2. 页面出现，向上滚动展示照护摘要内容
3. 继续滚动至「你可能想问医生的问题」清单
4. 继续滚动至绿色「生成可复制摘要」按钮
5. （可选）点击按钮，展示可复制摘要弹层

**Step 7**（45–50s）：字幕卡，后期制作

## 后期剪辑建议

### 字幕
- 使用 16px 以上白色字幕，深色半透明背景条
- 字幕显示时间：每句 2–3 秒
- Scene 1 和 Scene 7 做成文字动画卡

### 配音（可选）
- 语速适中，温柔但清晰
- 避免过于"广告腔"或"科技感"
- 可用真实照护者视角口吻："我把今天发生的事……"

### 背景音乐
- 轻柔钢琴 / 轻音乐，不喧宾夺主
- 音量约为配音的 20–30%
- 推荐曲风：lo-fi piano, gentle acoustic

### 转场
- 页面间切换：使用淡入淡出（0.3s）
- 避免：翻页、缩放、弹跳等夸张转场

### 时长控制
- 目标：40–50 秒
- 最长不超过 60 秒
